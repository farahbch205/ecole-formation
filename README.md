# Ecole de Formation Management System

## Description
Web application for managing a training school.
The system allows the administrator to manage students, teachers, courses and attendance.

## Technologies
- PHP
- MySQL
- HTML / CSS / JavaScript
- XAMPP

## Features
- Admin authentication
- Manage students
- Manage teachers
- Manage courses
- Attendance management

## Installation
1. Clone the project
2. Import database.sql in phpMyAdmin
3. Configure config.php
4. Run the project on localhost
